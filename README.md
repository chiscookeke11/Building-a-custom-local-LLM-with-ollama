## Introducing Today's Project!

Building My First Local AI Assistant with Ollama

### Key tools and concepts

This project introduced me to the fundamentals of running AI models locally using Ollama and open-source models from Hugging Face. Along the way, I learned about large language models (LLMs), model sizes and quantization, prompt engineering, custom AI personas using Modelfiles, and the basics of Retrieval-Augmented Generation (RAG) for improving AI responses with external knowledge.

### Challenges and wins

One of the biggest challenges was understanding how local AI models differ from cloud-based models, particularly their knowledge limitations and hardware requirements. I also encountered a few setup and configuration issues while working with Ollama, but troubleshooting them helped me better understand the workflow. Successfully running my first local model, customizing its behavior with a Modelfile, and learning the foundations of RAG were rewarding milestones that boosted my confidence in building AI application
